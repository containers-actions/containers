# LibreOffice NoGUI Container Image

LibreOffice headless (no GUI) container for document processing.

## Tags

- `latest`
- Version-specific tags

## Usage

```bash
docker run --rm -it ghcr.io/containers-actions/libreoffice-nogui:latest soffice --version
```

## Build Information

- Base Image: bitnami/os-shell
- Install Path: /usr/local/bin
