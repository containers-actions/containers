# FNM Container Image

Fast Node Manager (FNM) - A Node.js version manager built in Rust.

## Tags

- `latest`
- Version-specific tags

## Usage

```bash
docker run --rm -it ghcr.io/containers-actions/fnm:latest fnm --version
```

## Build Information

- Base Image: bitnami/os-shell
- Install Path: /usr/local/bin
